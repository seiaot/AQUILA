## ゲーム概要
奥行きのあるシューティングゲーム

元々スプライトを用いた2Dライブラリで3Dっぽいものを作れないかと思い作成開始
教員からの助言により、スペースハリアーを参考に

## 担当箇所
個人制作の為全て

## 制作上での（仮）

### 拡大縮小・移動
- 敵や雲は共通の基底クラスを使用しており、プレイヤーとの奥行き軸での距離によって位置と拡大率が変化するようにしている

### 当たり判定
- 元々存在していた矩形と円の当たり判定から、3D空間で使用可能なOBBと球の当たり判定を作成して変更

### 三角関数
- 最後に出現する敵の動きに三角関数を使用
- 制作に使用することで、それまで理解していなかった三角関数をある程度理解できた
